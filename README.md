# ï£¿ macOS-Style Interactive Portfolio

A premium, interactive desktop environment built with **React**, **GSAP**, and **Tailwind CSS**. This portfolio doesn't just display projects; it behaves like a fully functional operating system with draggable windows, a physics-based dock, and smooth system animations.

##  Features

- **macOS Desktop Environment**: Fully interactive desktop UI.
- **GSAP Animations**: Fluid, high-performance motion design and hover effects.
- **Draggable Windows**: Multi-window system with Z-index management (focusing).
- **Functional Apps**:
  - **Finder**: Navigate through project categories and folders.
  - **Terminal**: A CLI-themed view of your tech stack.
  - **Safari**: A browser-like interface for blog posts.
  - **Resume Viewer**: Integrated PDF viewer with a one-click download.
  - **Responsive Dock**: A magnifying dock with real-time physics and tooltips.
  - **Global State**: Built with **Zustand** and **Immer** for seamless window and location management.

##  Tech Stack

- **Framework**: [React](https://reactjs.org/) (Vite)
- **Animation**: [GSAP](https://greensock.com/gsap/) (GreenSock Animation Platform)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **State Management**: [Zustand](https://github.com/pmndrs/zustand) & [Immer](https://immerjs.github.io/immer/)
- **Icons**: [Lucide React](https://lucide.dev/)
